# Commonjs fix exports

cli tool to fix commonjs exports from exports.default to module.exports in a file or directory of files.

## Install

```bash
npm install -g cjs-fixer
```

## Usage

### fot a file

option is:

`-f` or `--file`

```bash
cjs-fixer -f ./path/to/file.js
```

### for a directory

option is:

`-d` or `--dir`

```bash
cjs-fixer -d ./path/to/dir
```
